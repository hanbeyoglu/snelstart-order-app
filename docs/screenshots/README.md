# UI screenshots

Replace the minimal placeholder PNGs in this folder with real application screenshots for the main `README.md` gallery.

**Suggested filenames** (already referenced in the root README):

| File | Suggested capture |
|------|-------------------|
| `dashboard.png` | Staff dashboard / analytics overview |
| `customer-portal.png` | Customer portal home or catalog |
| `products.png` | Product listing or detail |
| `cart.png` | Cart / checkout flow |
| `reports.png` | Reports & charts |
| `audit-logs.png` | Audit log table |
| `mobile-view.png` | Narrow viewport or device frame |

Export at **1280×720** or **1920×1080** for consistent GitHub rendering. Use PNG or WebP.
